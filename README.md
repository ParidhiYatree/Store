# Store
•	Extracted and prepared data using SQL and developed a multi-page Power BI dashboard to analyze sales, profit, customer segments, product performance, and shipping metrics.
•	Created DAX measures to track KPIs such as Total Sales, Total Profit, Items Sold, and Shipping Cost Analysis.
•	Built interactive visualizations to analyze regional sales trends, customer insights, product category performance, and order fulfillment efficiency.
•	Performed data cleaning, transformation, and data modeling for reliable reporting.
Outcome: Identified high-revenue regions, top-performing products, and cost-efficient shipping methods, supporting improved business and logistics decisions.
Tools: SQL, Power BI, DAX, Data Modeling, Data Visualization
